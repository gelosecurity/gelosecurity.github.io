Hypic WebView JavaScript Interface PoC

This page attempts to interact with exposed JavaScript interfaces in the Hypic Android app.

The target is the custom retouchOversea bridge which bypasses the standard Bytedance AuthConfig.

1. Test getUserInfo (Account Takeover)

This attempts to exfiltrate the session and user data via the custom bridge.

2. Test openSchema (Intent Redirection)

This attempts to launch a protected, non-exported internal Activity.

3. SDK Diagnostic Check

Checks if the standard Bytedance SDK bridge is present.
